NaturaEsencia - E-commerce de Cosmética Natural
**Proyecto Final: Portafolio Profesional - Python Fullstack Developer**

## Descripción
Este proyecto es una plataforma de e-commerce funcional desarrollada con **Django**. Permite la gestión de un catálogo de productos naturales, un sistema de carrito de compras dinámico y un flujo completo de autenticación de usuarios.

## 🛠️ Tecnologías Utilizadas
**Backend:** Python 3.x / Django
**Frontend:** HTML5, CSS3, Bootstrap 5 (Diseño Responsivo)
**Base de Datos:** SQLite (incluida para evaluación)
**Control de Versiones:** Git y GitHub

## 🚀 Instalación y Uso Local
1. Clonar el repositorio.
2. Crear entorno virtual: `python -m venv env`.
3. Activar entorno: `.\env\Scripts\activate`.
4. Instalar dependencias: `pip install django`.
5. Ejecutar: `python manage.py runserver`.

## ✨ Buenas Prácticas Aplicadas
**Seguridad:** Implementación de cierre de sesión mediante método POST para prevenir ataques CSRF.
**UX/UI:** Mejora de la jerarquía visual en el Navbar y optimización del contraste en el botón del carrito.
**Orden:** Uso de archivo `.gitignore` para mantener un repositorio limpio de archivos temporales.
